# Development Resources

This directory contains development resources for the project.

## Local Development Setup

Resources for local development work can be found in the `./.dev` directory:
- `schema.ts`: The main database schema definition
- `journals/`: Contains journal entries and documentation (.md files with development notes)

For detailed development logs and session tracking, see the journals directory which contains:
- Individual journal entries documenting development sessions
- Journal template and scripts for session documentation
- System documentation and usage guidelines
